하이퍼링크 삽입하기

내 PC내 하이퍼링크

a를 누르고 엔터
그러면 자동으로 href = ""가 생성됨
그 때 "_"안에 원하는 파일, 링크를 입력해준다
주의할 점 a와 /a 사이에 하이퍼링크 텍스트 입력

다른 페이지 하이퍼링크 삽입하기

마찬가지로 a로 열어줌

  1. href=" " 사이에 원하는 홈페이지 주소를 입력
  2. target="_"사이에 _blank를 입력 : 이는 곧 새창에서 열기를 뜻함
  3. title="_"사이에 html5 specification을 입력: 사실 이건 뭔지 잘 모르겠다. 더 알아보기
